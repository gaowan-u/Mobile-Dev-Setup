## 🛠️ 环境要求
- Android 7 及以上版本
- Termux 应用需更新至最新版
- 至少 500MB 可用存储空间
- 稳定的网络连接

## 🚀 一键安装方案
复制链接至Termux终端，然后按回车。
```bash
curl -L https://github.com/gaowan-u/Mobile-Dev-Setup/releases/download/install.sh/install.sh | bash
```
